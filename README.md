# ScreenMonitor

## Hotkeys
| Key      | Action                       |
| -------- | ---------------------------- |
| Up       | Scale window up              |
| Down     | Scale window down            |
| Left     | Next screen                  |
| Right    | Previous screen              |
| PageUp   | Higher refresh rate (slower) |
| PageDown | Lower refresh rate (faster)  |
| T        | Toggle TopMost               |
